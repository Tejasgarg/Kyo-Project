# Kyo-Project

The fully autonomous air purifier
* Made on the base of custom made 3d printed PLA chassis
* The Air purifier has wheels that allow it move to areas of high concentration of particulate matter
* It has array of IR sensors to navigate, PM 2.5 Sensor to measure air quality that is displayed on LCD screen in real time
